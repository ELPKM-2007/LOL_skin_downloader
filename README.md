# LOL_skin_downloader
You can use it to download all skins of a certain character in LOL

The exe file can be run directly.
The skin_for_hero folder is the fundamental Scrapy source code.
skin_downloader.py is the source code corresponding to the exe file (directly generated by AI based on the above file).

exe文件可以直接运行使用。
skin_for_hero文件夹是最根本的scrapy源码
