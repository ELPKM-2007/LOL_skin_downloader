# LOL_skin_downloader
You can use it to download all skins of a certain character in LOL
The exe file in the dist folder can be run directly.
The skin_for_hero folder is the fundamental Scrapy source code.
skin_downloader.py is the source code for the request logic (directly generated by AI based on the above file).
skin_downloader1.py is the source code of Scrapy compressed into a single py file (directly generated by AI based on the above file).
